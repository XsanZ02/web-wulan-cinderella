# Web Wulan Cinderella

Website statis yang dibuat sebagai cerita romantis dan hadiah digital untuk Wulan, dengan tampilan modern, gallery foto, mystery box, dan pemutar musik lokal.

## Fitur

- Hero section dengan narasi dan animasi bunga
- Gallery foto dengan lightbox
- Mystery box interaktif
- Music player MP3 lokal
- Desain responsif untuk desktop dan mobile

## Struktur proyek

- `index.html` — struktur utama halaman
- `styles.css` — styling UI dan animasi
- `script.js` — logika interaktif, music player, gallery, mystery box
- `music/` — file audio lokal
- `galery/` — file gambar gallery
- `flowers/` — aset bunga/efek animasi

## Cara menjalankan

Buka file `index.html` di browser langsung, atau jalankan server statis sederhana seperti:

```bash
python -m http.server 8000
```

Lalu buka:

```text
http://localhost:8000
```

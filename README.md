# Web Wulan Cinderella

Website statis yang dibuat sebagai hadiah digital untuk Wulan, dengan tampilan romantis, galeri foto, mystery box interaktif, dan pemutar musik lokal.

## Deskripsi

Proyek ini adalah halaman web single-page yang menghadirkan cerita personal dengan gaya elegan, animasi lembut, dan pengalaman interaktif yang membuat situs terasa lebih hidup.

## Fitur utama

- Hero section dengan narasi dan visual yang khas
- Galeri foto dengan lightbox
- Mystery box yang memunculkan efek bunga animasi
- Music player MP3 lokal
- Desain responsif untuk desktop dan mobile
- Tanpa proses build atau dependency tambahan

## Struktur folder

- `index.html` — struktur utama halaman
- `styles.css` — tata letak, warna, animasi, dan styling visual
- `script.js` — logika interaksi, galeri, musik, dan mystery box
- `music/` — file audio lokal
- `galery/` — koleksi gambar
- `flowers/` — aset visual bunga untuk efek animasi

## Cara menjalankan secara lokal

Anda bisa langsung membuka `index.html` di browser, atau menjalankan server statis sederhana:

```bash
python -m http.server 8000
```

Lalu buka:

```text
http://localhost:8000
```
